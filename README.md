# Real-Time Server Log Analysis for Attack Detection and Source Attribution

## Overview

This project is a real-time cybersecurity monitoring platform designed to collect, process, and analyze server logs for detecting cyber attacks and identifying their sources.

The system continuously monitors authentication logs, system logs, and web server logs to detect suspicious activities and generate security alerts for incident response and digital forensic investigations.

---

## Features

- Real-time server log monitoring
- SSH brute-force attack detection
- Port scanning detection
- SQL Injection detection
- Cross-Site Scripting (XSS) detection
- Unauthorized login detection
- Attack source attribution
- Security event correlation
- Incident reporting
- Security dashboard
